```uml
@startuml

ユーザー -> webサーバー : 通る道の入力
webサーバー --> ユーザー : 入力された道の表示

opt 途中で作成を中断する場合

alt 別画面に映ろうとしたとき
ユーザー -> webサーバー : 画面移動
webサーバー --> ユーザー : 保存の要否のポップアップ

alt 保存しない場合
ユーザー -> webサーバー : 保存しないを押す
webサーバー --> ユーザー : 遷移先ページの表示
end

else 保存する場合
ユーザー -> webサーバー : 保存するを押す
webサーバー --> ユーザー : 遷移先ページの表示
webサーバー -> DBサーバー : 作成データの保存を要求
DBサーバー --> DBサーバー : 作成データの保存処理
end


@enduml
```
