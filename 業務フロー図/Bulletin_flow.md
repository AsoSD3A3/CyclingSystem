```
@startuml

opt 交流掲示板を使用するとき

ユーザー -> webサーバー : 交流掲示板の選択
webサーバー -> DBサーバー : レス一覧の要求
DBサーバー --> webサーバー : レス一覧の送信
webサーバー --> ユーザー : レス一覧の表示
ユーザー -> webサーバー : チャットの入力
webサーバー -> DBサーバー : チャットの送信
DBサーバー -> DBサーバー : チャットの保存
DBサーバー --> webサーバー : 保存済みチャットの送信
webサーバー --> ユーザー : 保存済みチャットの表示

opt ルートの共有をするとき

ユーザー -> webサーバー : (date)から共有したいルートの選択
webサーバー -> DBサーバー : ルート詳細の要求
DBサーバー -> webサーバー : ルート詳細の送信
webサーバー -> ユーザー : ルート詳細の表示
ユーザー -> webサーバー : コメントをつけてshareを押す
webサーバー -> DBサーバー : コメント込みのルート詳細の送信
DBサーバー -> DBサーバー : 送信されたルートをpostに保存
DBサーバー -> webサーバー : 保存した情報の送信
webサーバー -> ユーザー : postページに保存したデータの表示
end

else 募集するとき

ユーザー -> webサーバー : メニューから募集を選ぶ

opt 募集するとき

ユーザー -> webサーバー : 募集するを選ぶ
webサーバー -> DBサーバー : 保存ルートの要求
DBサーバー --> webサーバー : 保存ルートの送信
webサーバー --> ユーザー : 保存ルートの表示
ユーザー -> webサーバー : 行きたいコースを選ぶ
webサーバー -> DBサーバー : 選択されたルートの要求
DBサーバー -> webサーバー : 選択ルートの送信
webサーバー --> ユーザー : カレンダーの表示
ユーザー -> webサーバー : 日付選択
webサーバー --> ユーザー : 選択ルートの表示
ユーザー -> webサーバー : 募集要項入力
webサーバー -> DBサーバー : 募集要項送信
DBサーバー -> DBサーバー : 募集要項保存and専用掲示板の作成
DBサーバー --> webサーバー : 作成した専用掲示板を送信
webサーバー --> ユーザー : 専用掲示板の表示

else 募集を見るとき

ユーザー -> webサーバー : 参加するを選ぶ
webサーバー -> DBサーバー : 募集一覧の要求
DBサーバー --> webサーバー : 募集一覧の送信
webサーバー --> ユーザー : 募集一覧の表示
ユーザー -> webサーバー : 参加するルートを選択
webサーバー -> DBサーバー : 専用掲示板を要求
DBサーバー -> DBサーバー : 参加要項保存
DBサーバー --> webサーバー : 専用掲示板を送信
webサーバー --> ユーザー : 専用掲示板の表示
end
end


@enduml
```
