```
@startuml

opt 交流掲示板をソートするとき

ユーザー -> webサーバー : 交流掲示板の選択
webサーバー -> DBサーバー : レス一覧の要求
DBサーバー --> webサーバー : レス一覧の送信
webサーバー --> ユーザー : レス一覧の表示
ユーザー -> webサーバー : チャットの入力
webサーバー -> DBサーバー : チャットの送信
DBサーバー -> DBサーバー : チャットの保存
DBサーバー --> webサーバー : 保存済みチャットの送信
webサーバー --> ユーザー : 保存済みチャットの表示

opt ルートの共有をするとき

ユーザー -> webサーバー : 共有URLをコピー
ユーザー -> webサーバー : 共有URLを共有したいレスに貼り付け
end

else 募集掲示板をソートするとき

ユーザー -> webサーバー : 募集掲示板の選択

opt 募集するとき

ユーザー -> webサーバー : 共有URLと募集要項を入力してスレ立て
webサーバー -> DBサーバー : 入力された情報の送信
DBサーバー -> DBサーバー : 入力された情報の保存
DBサーバー -> DBサーバー : スレ情報を一覧に追加

else 募集を見るとき

webサーバー -> DBサーバー : 募集一覧の要求
DBサーバー --> webサーバー : 募集一覧の送信
webサーバー --> ユーザー : 募集一覧の表示
end
end


@enduml
```
